## 2026-01-11 - "The Ghost in the Machine" (Copy Logs)
**Learning:** UX Memory/Documentation can drift from reality. The "Memory" system stated the EventLog had a copy feature, but the code (Source of Truth) did not. Always verify "existing" features before assuming they are there.
**Action:** When a task involves "improving" a feature based on description, first verifying its existence with a reproduction script is critical. This turns a "fix" into a "feature add" which clarifies the scope.

## 2026-01-11 - Delightful Micro-Interactions
**Learning:** Adding small, high-utility features like "Copy Logs" with immediate visual feedback (Checkmark) creates a sense of polish and responsiveness that static lists lack.
**Action:** Look for other read-only data lists (like Trade History) that could benefit from export/copy actions to empower power users.
